Some of my recent projects:

  • typr

    Ruby on Rails + Redis, CoffeeScript

    Typr is a competitive and social typing application. I began developing Typr in order to explore technologies like Devise and Omniauth, and it led me to discover some amazing new ideas, namely Redis and CoffeeScript.

    Features

    • Meticulous tracking and analysis of statistics over time
    • Social features: multi-player competition mode, in which users "race" to finish typing
    • Speedy game client, written in CoffeeScript
    • Real-time game server, performing player matchmaking and synchronizing player statistics, using Redis

  • jido

    Ruby, Redis, Sinatra, Nokogiri

    Jido is a ridiculously simple and fast verb conjugation library. Backed by an XML store of verb analyses for a growing number of languages, Jido generates verb conjugations for each and every possible linguistic tense, mood, and paradigm in under one-tenth of a second.

    Features

    • Completely dynamic, modular backend: Any language's basic grammatical rules can be converted to Jido's data format and then immediately be used to generate regular and irregular verb conjugations
    • Sinatra-based front-end, which conjugates, caches and displays verbs
